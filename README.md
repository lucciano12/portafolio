# Portafolio — Lucciano Prado

Portafolio personal construido con Next.js 14 y Tailwind CSS.

🌐 **Live:** [lucciano12.vercel.app](https://lucciano12.vercel.app)

## Stack
- Next.js 14
- Tailwind CSS
- TypeScript

## Correr localmente

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000)
